How to Activate Virt in Git Bash Terminal 
source virt/Scripts/activate


How to Save Git Hub Paste the 3 lines below in the Git Bash Terminal 
git add . 
git commit -am'tweaked ______'  
______ = file name ex: Flask_Activate.txt (Make sure to save before entering the code)
git push 

Initializing Git For Version Control
$ git config --global user.name "Your Name"
$ git config --global user.email "you@youraddress.com"
$ git config --global push.default matching
$ git config --global alias.co checkout
$ git init
